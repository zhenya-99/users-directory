# Users Directory

A simple responsive web application that fetches and displays user data from an external REST API.

The application dynamically renders user cards using JavaScript and handles loading and error states.

---

## 🚀 Features

- Fetch users from external REST API
- Dynamic rendering of user cards
- Loading state while fetching data
- Error handling for failed requests
- Select number of users to display
- Responsive layout for different screen sizes

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- REST API
- DOM Manipulation

---

## 🌐 API

This project uses JSONPlaceholder:

https://jsonplaceholder.typicode.com/users

---

## ⚙️ How It Works

1. User selects how many users to display
2. Clicks "Show" button
3. Application sends request to API
4. Data is received and rendered as user cards
5. Loading and error states are handled accordingly
